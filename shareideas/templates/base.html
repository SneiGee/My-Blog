LastoftheDinosaurs

GitHub Twitch YouTube Email

Hello, I'm Last, a retro variety streamer, amateur coder, and long-time Linux user. I made this website to learn new skills while sharing my hobbies. This website is open source and created with Python, Flask, Bootstrap, and more. Check my GitHub page for complete source code.

I collect third through sixth-gen retro games and consoles, intending to own a copy of every North American Sega Saturn game (currently 81 out of 258). All of the games I stream are played on real hardware; 518 games across ten consoles and nearly every genre.

I've been using Linux since the late 90s, ever since shortly after Red Hat 6.0 was released (not the Enterprise edition, this release was the first to introduce glibc, and it's also when GNOME was officially integrated). I eventually made a career out of what was and still is a favorite hobby.
